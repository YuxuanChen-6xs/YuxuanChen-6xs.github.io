## 避坑指南——常见报错与解决方法

新手接入 DeepSeek 时最容易犯的几个错误，我列出来帮你避雷：

### 错误1：`base_url` 拼写错误

很多人写成 `base_url="https://api.deepseek.com/v1"`，但千聚的接口地址必须是 `https://www.qianjuai.com/v1`。DeepSeek 原生 API 在国内访问不稳定，千聚做了优化，所以必须用千聚地址。

### 错误2：`api_key` 用了 DeepSeek 官方的 Key

你的 API Key 必须是从千聚控制的台创建的。如果用了 DeepSeek 官方申请的 Key，报错信息会是 "Incorrect API key"。

**解决方法：** 从千聚重新创建 Key，替换即可。

DeepSeek-R1 的模型名是 `deepseek-r1`，DeepSeek-V3 是 `deepseek-v3`。大小写敏感，别写错了。

**常见错误：** `DeepSeek-R1`、`deepseek R1` 都会报错。正确写法：`deepseek-r1`。
