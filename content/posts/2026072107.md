### 第三步：编写调用代码（完整可复制）

创建一个类 `ChatGPTClient.java`，直接贴入以下代码：

java
import okhttp3.*;
import com.google.gson.JsonObject;
import com.google.gson.JsonParser;
import java.io.IOException;

public class ChatGPTClient {
    // 千聚AI聚合平台的 api base url
    private static final String BASE_URL = "https://www.qianjuai.com/v1";
    // 你的 API Key
    private static final String API_KEY = "sk-xxxx"; // 替换为你的 key

    public static void main(String[] args) throws IOException {
        // 构造请求体
        JsonObject requestBody = new JsonObject();
        requestBody.addProperty("model", "gpt-3.5-turbo");
        requestBody.addProperty("max_tokens", 200);

        JsonObject message = new JsonObject();
        message.addProperty("role", "user");
        message.addProperty("content", "你好，用中文回复：Java接入OpenAI示例");
        requestBody.add("messages", new JsonParser().parse("[{" + message.toString() + "}]").getAsJsonArray());

        OkHttpClient client = new OkHttpClient();
        MediaType JSON = MediaType.get("application/json; charset=utf-8");
        RequestBody body = RequestBody.create(JSON, requestBody.toString());

        Request request = new Request.Builder()
                .url(BASE_URL + "/chat/completions")
                .addHeader("Authorization", "Bearer " + API_KEY)
                .addHeader("Content-Type", "application/json")
                .post(body)
                .build();

        try (Response response = client.newCall(request).execute()) {
            if (!response.isSuccessful()) {
                System.out.println("请求失败: " + response.code());
                return;
            }
            String responseBody = response.body().string();
            // 简单解析响应
            JsonObject jsonResponse = JsonParser.parseString(responseBody).getAsJsonObject();
            String reply = jsonResponse.getAsJsonArray("choices")
                    .get(0).getAsJsonObject()
                    .get("message").getAsJsonObject()
                    .get("content").getAsString();
            System.out.println("ChatGPT回复: " + reply);
        }
    }
}


> 🔥 **说明**：代码里使用了 `gpt-3.5-turbo`，如果你想用 GPT-4 或其他模型，直接改 `model` 字段就行。千聚API支持 500+ 模型，随时切换。
