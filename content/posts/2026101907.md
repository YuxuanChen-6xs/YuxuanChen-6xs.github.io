### 安装依赖

因为千聚接口兼容OpenAI格式，我们直接用官方的`openai` npm包：

### 基础调用代码

javascript
const OpenAI = require('openai');

const client = new OpenAI({
  apiKey: '你的千聚API Key', // 替换为真实Key
  baseURL: 'https://www.qianjuai.com/v1'
});

async function testKimiK2() {
  try {
    const completion = await client.chat.completions.create({
      model: 'kimi-k2-thinking', // 注意模型名，千聚上实际叫这个（请以官方列表为准）
      messages: [
        { role: 'user', content: '解释一下什么是快速排序，并用JavaScript实现。' }
      ],
      temperature: 0.7,
      max_tokens: 1024
    });
    console.log(completion.choices[0].message.content);
  } catch (err) {
    console.error('请求失败:', err.message);
  }
}

testKimiK2();


### 流式输出（SSE）版本

async function streamKimiK2() {
  const stream = await client.chat.completions.create({

    stream: true
  });

  for await (const chunk of stream) {
    process.stdout.write(chunk.choices[0]?.delta?.content || '');
  }
}

## 避坑指南（我已经替你踩完了）

1. **模型名称一定要对**  
   千聚API上的模型名可能和官方略有不同。请先在千聚的模型列表页（控制台内可查）确认Kimi K2 Thinking的具体名称。我测试时使用的是`kimi-k2-thinking`，但后续可能更新，务必以官方最新为准。

2. **API Key不能复制错**  
   复制Key时注意不要遗漏字符或包含空格，否则会报401错误。

3. **注意baseURL的结尾**  
   `https://www.qianjuai.com/v1` 后面不要加多余的斜杠。

4. **余额不足会报错**  
   新用户免费额度用完后，最低充1元即可继续。代码中建议捕获`insufficient_quota`错误并提示用户充值。

5. **超时设置**  
   对于长思考模型，建议设置较长的超时时间（如30秒）。使用`openai`包时可以通过`timeout`参数：

   javascript
   const client = new OpenAI({
     apiKey: '...',
     baseURL: 'https://www.qianjuai.com/v1',
     timeout: 30000
   });
   

6. **生产环境不要硬编码Key**  
   请将API Key放入环境变量，例如`process.env.QIANJU_API_KEY`。

## 进阶用法：多轮对话 & 函数调用

千聚API完全支持OpenAI格式的函数调用(function calling)，你可以让Kimi K2 Thinking调用你自己定义的函数。以下是一个简单的示例：

javascript
const tools = [
  {
    type: 'function',
    function: {
      name: 'get_weather',
      description: '获取指定城市的天气',
      parameters: {
        type: 'object',
        properties: {
          city: { type: 'string' }
        },
        required: ['city']
      }
    }
  }
];

const res = await client.chat.completions.create({
  model: 'kimi-k2-thinking',
  messages: [{ role: 'user', content: '北京今天天气如何？' }],
  tools: tools
});

console.log(res.choices[0].message);
