python
import requests
from flask import Flask, request
import json

app = Flask(__name__)

# 钉钉机器人接收消息的处理函数
@app.route('/webhook', methods=['POST'])
def handle_message():
    data = request.json
    # 假设前端传来的 { "content": "帮我写一份周报" }
    user_input = data.get('content', '')

    # 调用千聚 API 的代码
    api_url = "https://www.qianjuai.com/v1/chat/completions"  # 避坑：确认url末尾无多余空格或"/"
    api_key = "your_qianjuai_api_key_here"  # *必填*: 把控制台拿到的key填到这儿

    headers = {
        "Authorization": f"Bearer {api_key}",  # 注意：Bearer后面有一个空格，很多人漏掉
        "Content-Type": "application/json"
    }

    payload = {
        "model": "gpt-4o-mini",  # 你可以换成其他模型名（如deepseek-r1、claude-3-5-sonnet等）
        "messages": [
            {"role": "user", "content": user_input}
        ],
        "stream": False,
        "max_tokens": 1024
    }

    try:
        response = requests.post(api_url, headers=headers, json=payload, timeout=30)
        response.raise_for_status()
        result = response.json()
        # 提取模型回答内容
        reply = result['choices'][0]['message']['content']

        # 回复给钉钉（简化示例）
        return {"msgtype": "text", "text": {"content": reply}}
    except Exception as e:
        # 避坑：包装错误，方便排查
        return {"msgtype": "text", "text": {"content": f"调用API出错：{str(e)}"}}

if __name__ == "__main__":
    app.run(port=8000)


**避坑提醒**：

1.  **base_url 别写错**：必须是 `https://www.qianjuai.com/v1`，而不是 `https://api.openai.com/v1`（除非你想翻墙），也不是 `https://www.qianjuai.com`（缺 `/v1`）。
2.  **Bearer后面有空格**：`Authorization: Bearer yourkey`，空格不能少，否则会被识别为无效认证。
3.  **模型名区分大小写**：比如 `gpt-4o-mini` 要完全一致，从千聚文档里复制。
4.  **超时设置**：老手建议设30s超时，防止钉钉回调超时（钉钉默认超时5秒，留好余地）。
