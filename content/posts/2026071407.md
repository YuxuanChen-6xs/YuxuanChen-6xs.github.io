## Java实战：快速接|入GPT-5-thinking

下面我们直接上干货。假设你要在一个新项目里接入GPT-5-thinking，使用最流行的`OpenAI Java SDK`，整个流程极其丝滑。

### 1. 引入Maven依赖

在你的`pom.xml`里，添加官方的OpenAI Java库依赖（推荐最新稳定版）。

<dependency>
    <groupId>com.theokanning.openai-gpt3-java</groupId>
    <artifactId>service</artifactId>

</dependency>


### 2. 配置客户端（核心一步）

这是整个接入过程中，你需要“动脑子”的唯一操作：修改`base_url`。

java
import com.theokanning.openai.OpenAiService;
import com.theokanning.openai.completion.chat.ChatCompletionRequest;
import com.theokanning.openai.completion.chat.ChatMessage;
import okhttp3.OkHttpClient;
import retrofit2.Retrofit;

import java.time.Duration;
import java.util.List;

public class GPT5ThinkingExample {

    public static void main(String[] args) {
        // 你的千聚API Key
        String apiKey = "your-qianju-api-key-here";
        
        // 核心配置：将base_url指向千聚API的v1端点
        String baseUrl = "https://www.qianjuai.com/v1/"; 
        // 注意：有些版本库末尾必须要有斜杠

        // 创建OpenAiService实例
        // 这里只需要传入baseUrl和apiKey，以及超时时间即可
        OpenAiService service = new OpenAiService(baseUrl, apiKey, Duration.ofSeconds(60));

        // 构建聊天请求
        ChatCompletionRequest request = ChatCompletionRequest.builder()
                .model("gpt-5-thinking") // 直接指定模型名称
                .messages(List.of(
                        new ChatMessage("system", "你是一个资深的Java专家。"),
                        new ChatMessage("user", "请用Java写一个简单的示例代码，演示如何调用你的API。")
                ))
                .maxTokens(2048)
                .build();
                
        // 发送请求并打印结果
        service.createChatCompletion(request).getChoices().forEach(choice -> {
            System.out.println(choice.getMessage().getContent());
        });
        
        // 如果你需要流式（Streamin）响应，方法也完全一样
         // service.streamChatCompletion(request)...
    }
}
