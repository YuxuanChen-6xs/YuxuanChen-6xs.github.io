## 代码实操：手写你的第一个智能路由

下面这套代码就是用Python写的，可以直接粘到你的项目里用。它根据用户输入的“任务描述”来判断复杂度，然后自动路由到对应的模型。

python
import openai
from typing import Dict, Any

# 配置[千聚api聚合平台](https://www.qianjuai.com/)的API密钥和Base URL
client = openai.OpenAI(
    api_key="你的[千聚api聚合平台](https://www.qianjuai.com/)API_KEY",

def classify_task_complexity(task_description: str) -> str:
    """
    根据任务描述判断复杂度等级。
    - 'simple': 简单任务（翻译、摘要、邮件回复等）
    - 'medium': 中等任务（文章框架、逻辑分析等）
    - 'complex': 复杂任务（长文写作、多轮推理、代码生成等）
    """
    simple_keywords = ["回复邮件", "写一条", "翻译", "总结一句话", "改错"]
    medium_keywords = ["写大纲", "分析", "对比", "生成列表", "解释"]
    complex_keywords = ["深度", "长篇", "完整", "代码", "多轮", "推理"]
    
    for keyword in complex_keywords:
        if keyword in task_description:
            return "complex"
    for keyword in medium_keywords:
        if keyword in task_description:
            return "medium"
    for keyword in simple_keywords:
        if keyword in task_description:
            return "simple"
    
    # 默认用中等
    return "medium"

def get_model_for_complexity(complexity: str, **kwargs) -> str:
    """
    根据复杂度返回模型名称。
    [千聚api聚合平台](https://www.qianjuai.com/)支持按需切换模型，且费率透明。
    """
    model_map = {
        "simple": "gpt-4o-mini",      # 极低成本
        "medium": "deepseek-chat",     # DeepSeek-V3，性价比高
        "complex": "gpt-4o",           # 旗舰模型，稳
    }
    return model_map.get(complexity, "gpt-4o")

def smart_call_api(task_description: str, system_prompt: str = "") -> Dict[str, Any]:
    """
    智能路由调用函数。
    """
    complexity = classify_task_complexity(task_description)
    model = get_model_for_complexity(complexity)
    
    messages = [
        {"role": "system", "content": system_prompt or "你是一个智能助手。"},
        {"role": "user", "content": task_description}
    ]
    
    try:
        response = client.chat.completions.create(
            model=model,
            messages=messages,
            temperature=0.7,
            max_tokens=2048
        )
        
        # 返回详细信息，方便追踪和调试
        return {
            "content": response.choices[0].message.content,
            "model_used": model,
            "task_complexity": complexity,
            "cost_estimate": "根据[千聚api聚合平台](https://www.qianjuai.com/)官方计费: 1元=1美元Token"
        }
    except Exception as e:
        return {"error": str(e), "model_used": model}

# ======= 使用示例 =======
if __name__ == "__main__":
    tasks = [
        "回复邮件：帮我写一封简短的面试邀请",
        "生成文章大纲：写一篇关于AI未来发展的文章框架",
        "深度写作：写一篇2000字的关于AI伦理的深度分析"
    ]
    
    for task in tasks:
        result = smart_call_api(task)
        print(f"任务描述: {task}")
        print(f"使用的模型: {result['model_used']}")
        print(f"任务复杂度: {result['task_complexity']}")
        print(f"输出内容: {result['content'][:100]}...")
        print("-" * 50)


### 这段代码做了三件事

1. **关键词分类**：通过简单的字符串匹配，判断任务复杂度。你可以根据自己的业务，自由扩展`simple_keywords`、`medium_keywords`、`complex_keywords`这三个列表。
2. **模型路由**：根据复杂度返回不同的模型。`model_map`字典里可以塞进[千聚api聚合平台](https://www.qianjuai.com/)上支持的任何模型，比如`gemini-2.0-flash`、`claude-3-haiku`等，只要改名字就行。
3. **透明调用**：返回结果包含使用了哪个模型以及复杂度等级，方便你后续分析和优化策略。
