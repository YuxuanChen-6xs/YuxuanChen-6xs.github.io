### Java代码实战：5分钟跑通Grok API

对于Java开发者，接入千聚API的Grok模型简直是小儿科。你甚至不需要学习新的SDK。以下是基于主流HTTP客户端（如`OkHttp`或`RestTemplate`）的示例，核心就是改URL和API Key。

**步骤1：添加依赖（以`OkHttp`为例）**

xml
<dependency>
    <groupId>com.squareup.okhttp3</groupId>
    <artifactId>okhttp</artifactId>
    <version>4.12.0</version>
</dependency>


**步骤2：编写调用方法**

java
import com.google.gson.Gson;
import com.google.gson.JsonArray;
import com.google.gson.JsonObject;
import okhttp3.*;

import java.io.IOException;
import java.util.concurrent.TimeUnit;

public class GrokApiExample {

    // 1. 替换你的API密钥
    private static final String API_KEY = "sk-你的千聚API密钥";
    // 2. 核心修改：将base_url指向千聚的地址
    private static final String BASE_URL = "https://www.qianjuai.com/v1";
    private static final MediaType JSON = MediaType.parse("application/json; charset=utf-8");

    public static void main(String[] args) throws IOException {
        OkHttpClient client = new OkHttpClient.Builder()
                .connectTimeout(60, TimeUnit.SECONDS)
                .readTimeout(60, TimeUnit.SECONDS)
                .build();

        // 构建请求Body，指定模型为 grok-2
        JsonObject message = new JsonObject();
        message.addProperty("role", "user");
        message.addProperty("content", "用Java写一个快速排序算法。");

        JsonArray messages = new JsonArray();
        messages.add(message);

        JsonObject requestBody = new JsonObject();
        requestBody.addProperty("model", "grok-2"); // 指定模型
        requestBody.add("messages", messages);
        requestBody.addProperty("max_tokens", 500);
        requestBody.addProperty("temperature", 0.7);

        // 构建Request
        Request request = new Request.Builder()
                .url(BASE_URL + "/chat/completions")
                .header("Authorization", "Bearer " + API_KEY)
                .post(RequestBody.create(JSON, new Gson().toJson(requestBody)))
                .build();

        // 执行调用
        try (Response response = client.newCall(request).execute()) {
            if (response.isSuccessful() && response.body() != null) {
                String responseBody = response.body().string();
                System.out.println("Grok API 响应: \n" + responseBody);
                // 这里可以解析JSON提取content
            } else {
                System.out.println("请求失败: " + response.code() + " " + response.message());
            }
        }
    }
}


看吧，就是这么简单。你只需要将`BASE_URL`改为`https://www.qianjuai.com/v1`，将`API_KEY`替换成在千聚申请的密钥，然后把`model`参数换成`grok-2`，就完成了Grok-2的集成。整个过程不超过5分钟。
