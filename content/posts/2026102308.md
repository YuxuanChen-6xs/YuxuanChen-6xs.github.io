## 手把手教程：四步搞定

准备好了吗？下面开始。整个过程大概持续 15-30 分钟，取决于你机器下载模型的速度。

### 第一步：安装 Ollama 并拉取 Embedding 模型

Ollama 是目前本地运行大模型的王者级工具，极简。去 [ollama.com](https://ollama.com) 下载对应系统的版本，双击安装即可。

安装完后，打开终端或命令行，拉取一个很适合中文且效果不错的 Embedding 模型——`bge-m3`（BAAI 出品，支持 8192 长度的上下文）：

bash

# 拉取 bge-m3 模型，大约 2GB，第一次运行自动下载

ollama pull bge-m3

下载完成后，你直接输入以下命令测试模型是否能正常使用：

bash

ollama run bge-m3

你能看到一个交互式命令行，说明本地模型启动成功了。按住 Ctrl+D 退出交互。

### 第二步：启动 Ollama 服务

默认情况下，Ollama 只监听在 `127.0.0.1:11434`。你需要确保服务一直运行着。

如果你安装后在后台没看到 Ollama 图标，可以手动启动：

bash

ollama serve

这样，本地就有一个 API 端点 `http://localhost:11434/api/embeddings`。但这个端点的格式不是 OpenAI 兼容的（它是 Ollama 自用格式），所以我们需要第三步来做转接。

### 第三步：搭建兼容接口代理层

这里我们借助[千聚ai聚合站](https://www.qianjuai.com/)社区提供的一个简单镜像 `qianjuai/embedding-proxy`，它能将 Ollama 的标准输出自动转换成 `v1/embeddings` 的官方格式。

在你的终端里执行以下 Docker 命令：

bash

docker run -d --name embedding-proxy \
  -p 8080:8080 \
  -e OLLAMA_BASE_URL=http://host.docker.internal:11434 \
  qianjuai/embedding-proxy:latest

> **说明**：
> - `-p 8080:8080` 表示把代理暴露在宿主机的 8080 端口。
> - `-e OLLAMA_BASE_URL` 告诉代理 Ollama 服务在哪里。在 Docker 容器里，`host.docker.internal` 指向宿主机 IP。
> - 如果你没有装 Docker，去 [docker.com](https://www.docker.com) 下载一个 Docker Desktop 安装即可，在 Windows/Mac 上很好用。

启动后，用 curl 快速验证接口是否正常工作：

bash

curl http://localhost:8080/v1/embeddings \
  -H "Content-Type: application/json" \
  -d '{
    "input": "千聚聚合AI开发平台很好用",
    "model": "bge-m3"
  }'

如果返回值类似：

json

{
  "object": "list",
  "data": [{
    "object": "embedding",
    "index": 0,
    "embedding": [0.01234, -0.05678, ...]  // 一串数字
  }],
  "model": "bge-m3",
  "usage": {
    "prompt_tokens": 8,
    "total_tokens": 8
  }
}

恭喜，你已经拥有了一个完全 OpenAI 兼容的 Embedding API 服务。

### 第四步：在代码里像调 OpenAI 一样调用它

现在把代码里的 `base_url` 从 `https://api.openai.com/v1` 改成 `http://localhost:8080/v1` 和你的任何 API Key（代理层其实不校验 key，你可以随便写一个）。

**Python 示例（用 openai 官方库）：**

python

from openai import OpenAI

client = OpenAI(
    base\_url="http://localhost:8080/v1",
    api\_key="sk-随便填一个, 因为本地不校验"
)

response = client.embeddings.create(
    model="bge-m3",  # 对应你Ollama里拉取的模型名
    input=["千聚聚合AI开发平台很好用", "本地搭建省钱"]
)

print(response.data[0].embedding)  # 打印第一个文本的向量

**换成 LangChain 也类似：**

python

from langchain\_community.embeddings import OpenAIEmbeddings

embeddings = OpenAIEmbeddings(
    model="bge-m3",
    openai\_api\_key="随便填",
    openai\_api\_base="http://localhost:8080/v1"
)

vector = embeddings.embed\_query("千聚聚合AI开发平台很好用")
print(vector)

你的整个 RAG 流水线不需要改任何业务逻辑，只需要改 `base_url` 这行。

**Q: 我的机器显卡显存不够怎么办？**

**Q: 这个接口支持并行请求吗？**
