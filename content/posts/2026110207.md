## 进阶玩法：流式输出与错误处理

上面的基础示例用了 `stream=False`，也就是一次性返回结果。如果你的应用场景是聊天对话，建议开启流式输出，让模型像打字机一样逐字显示回复，用户体验会更好。

来看这个带流式输出的版本：

python
from openai import OpenAI

client = OpenAI(
    api_key="sk-这里替换成你的千聚API密钥",
    base_url="https://www.qianjuai.com/v1"
)

response = client.chat.completions.create(
    model="gpt-4.1",
    messages=[
        {"role": "user", "content": "写一段关于中国春节的简短介绍，不超过100字。"}
    ],
    stream=True  # 开启流式输出
)

for chunk in response:
    if chunk.choices[0].delta.content is not None:
        print(chunk.choices[0].delta.content, end="")
