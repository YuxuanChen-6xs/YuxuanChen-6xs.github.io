### 代码解析（不感兴趣可以直接跳过）

如果你对代码本身好奇，简单说一下。`Gpt5ProController` 里用了 `OpenAiClient` 这个类：

java
@RestController
public class Gpt5ProController {

    @Autowired
    private OpenAiClient client;

    @GetMapping("/chat")
    public ChatCompletionResult chat(@RequestParam("message") String message) {
        ChatCompletionRequest request = ChatCompletionRequest.builder()
            .model("gpt-5pro")
            .messages(ChatMessage.builder()
                .role(ChatMessage.Role.USER)
                .content(message)
                .build())
            .build();
        return client.chatCompletion(request);
    }
}


注意 `ChatCompletionRequest` 的构建方式和旧版 OpenAI 一模一样，没有任何新参数。这就是我说的“兼容旧 API”——你把模型名改成 "gpt-5pro"，其他不用动。
