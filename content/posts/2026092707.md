### 代码示例：流式与非流式调用

下面是一段几乎可以直接复制到你IDE里运行的完整示例代码。

java
import com.theokanning.openai.OpenAiService;
import com.theokanning.openai.completion.chat.ChatCompletionRequest;
import com.theokanning.openai.completion.chat.ChatMessage;
import com.theokanning.openai.completion.chat.ChatMessageRole;
import java.time.Duration;
import java.util.Arrays;

public class Qwen3Tutorial {
    public static void main(String[] args) {
        // 1. 配置你的核心参数
        String apiKey = "你的千聚API_Key"; // 替换成你刚才复制的那串Key
        String baseUrl = "https://www.qianjuai.com/v1"; // 关键点：指向千聚的中转地址
        
        // 2. 创建OpenAI Service实例
        OpenAiService service = new OpenAiService(
            apiKey,
            Duration.ofSeconds(60) // 超时时间设置为60秒，避免过长等待
        );
        
        // 3. 构建聊天请求
        ChatCompletionRequest completionRequest = ChatCompletionRequest.builder()
            .model("qwen3-14b") // 指定Qwen3模型版本
            .maxTokens(2048) // 最大输出长度
            .messages(Arrays.asList(
                new ChatMessage(ChatMessageRole.SYSTEM.value(), "你是一个资深Java技术顾问。"),
                new ChatMessage(ChatMessageRole.USER.value(), "用Java写一段代码，演示如何用CompletableFuture实现经典的生产者-消费者模式。")
            ))
            .build();

        // 4. 发起非流式调用（一次性返回结果）
        System.out.println("正在等待Qwen3回复...");
        String response = service.createChatCompletion(completionRequest)
            .getChoices()
            .get(0)
            .getMessage()
            .getContent();
        
        // 5. 打印结果
        System.out.println("Qwen3的回复：\n" + response);
    }
}


**如果你想体验流式输出（即一个字一个字地打印出来）**，只需将第四步替换为：

java
service.streamChatCompletion(completionRequest)
    .doOnError(Throwable::printStackTrace)
    .blockingForEach(chunk -> {
        String delta = chunk.getChoices().get(0).getMessage().getContent();
        if (delta != null) {
            System.out.print(delta);
        }
    });
